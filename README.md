# PGCB Demand Forecasting Agile Data Science Project

This repository contains an Agile Data Science project for Bangladesh national grid electricity demand forecasting using the PGCB hourly generation dataset.

## Project Objective
The objective is to prepare data, build baseline and improved forecasting models, create validation scripts, and support dashboard-based decision-making.

## Main Components
- Data quality validation
- Feature engineering
- Baseline model
- Improved model
- Streamlit dashboard
- Automated testing with GitHub Actions

## Dataset
PGCB Hourly Generation Dataset (Bangladesh), UCI Machine Learning Repository.
